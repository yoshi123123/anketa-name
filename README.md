# ANKETA.FORUM

Форум с анкетами, ролями и админ-панелью. Node.js + Express + SQLite.

## Что внутри

- 🔐 Регистрация и вход через JWT в HTTP-only куки (сессия живёт 30 дней)
- 👥 Иерархия ролей: **owner** (главный админ) → **admin** → **moderator** → **user**
  - Главный админ создаётся автоматически из `.env` при первом запуске
  - Главный админ назначает/снимает админов и модераторов через UI
  - Админ может ставить только модеров (и не может тронуть других админов)
- 📋 Анкеты: создание, редактирование, удаление, закрепление, скрытие
- 💬 Форум: темы и ответы с теми же действиями
- 🛠 Live-редактор сайта: название, цвет акцента, приветствие, режим анонимности — всё меняется прямо из браузера, для всех пользователей сразу
- 🥷 Режим анонимности для админов: посты и комментарии показываются как «Анонимный админ»
- ⚡ Иконки управления (✏️ 📌 🙈 🗑️) появляются в углу анкеты при наведении для модера+
- 🚫 Бан/разбан пользователей с сохранением в БД
- 📜 Лог действий админов (кто, когда, что сделал)
- 💾 Реальная БД SQLite — данные сохраняются между перезагрузками

## Локальный запуск

Нужен Node.js 18+.

```bash
# 1. Установить зависимости
npm install

# 2. Создать .env (скопируй из примера и поменяй пароль)
cp .env.example .env
# открой .env и поменяй OWNER_PASSWORD и JWT_SECRET

# 3. Запустить
npm start
```

Открой <http://localhost:3000>. Войди под логином/паролем главного админа из `.env`.

## Структура

```
anketa-forum/
├── server.js          ← Express + SQLite + JWT + все API
├── package.json
├── .env.example       ← скопируй в .env и поменяй
├── public/
│   └── index.html     ← фронт (один файл, без сборки)
├── README.md
└── DEPLOY.md          ← как выложить на бесплатный хостинг
```

База данных создаётся автоматически в `db.sqlite` при первом запуске.
Чтобы стереть всё и начать с нуля — удали файл `db.sqlite` и перезапусти.

## API endpoints (кратко)

| Метод | Путь | Описание |
|---|---|---|
| POST | `/api/auth/register` | регистрация |
| POST | `/api/auth/login` | вход |
| POST | `/api/auth/logout` | выход |
| GET | `/api/auth/me` | текущий пользователь |
| PATCH | `/api/auth/me` | обновить свой профиль |
| POST | `/api/auth/change-password` | сменить пароль |
| GET | `/api/users` | список (модер+) |
| POST | `/api/users/:id/role` | назначить роль (админ+) |
| POST | `/api/users/:id/ban` | бан/разбан (модер+) |
| DELETE | `/api/users/:id` | удалить юзера (только owner) |
| GET/POST | `/api/profiles` | анкеты |
| PATCH/DELETE | `/api/profiles/:id` | редактирование/удаление |
| POST | `/api/profiles/:id/comments` | комментарий |
| GET/POST | `/api/topics` | темы форума |
| POST | `/api/topics/:id/posts` | ответ в теме |
| GET/PATCH | `/api/settings` | live-настройки сайта |
| GET | `/api/audit` | лог действий |

См. `DEPLOY.md` для деплоя на Render.com — бесплатно, ~10 минут.
